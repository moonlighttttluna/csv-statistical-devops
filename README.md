# 📊 CSV Statistical Analysis System (DevOps Integrated)

## 📌 Project Overview

This project is an automated CSV statistical processing system developed using Python.

The system:
- Automatically detects CSV files inside the `input/` folder
- Applies five statistical analysis functions
- Saves results in the `output/` folder
- Uses PyTest for automated testing
- Integrates Continuous Integration (CI) using GitHub Actions

---

## ⚙ Implemented Statistical Functions

The system includes the following functions:

1. calculate_column_mean
2. calculate_column_median
3. calculate_column_std_dev
4. generate_statistical_summary
5. correlation_analysis

Each function is implemented in a separate module inside the `processors/` folder.
